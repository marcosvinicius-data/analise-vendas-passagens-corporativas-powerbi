# ✈️ Análise de Vendas de Passagens Corporativas | Power BI

## 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de analisar o desempenho comercial de vendas de passagens corporativas entre janeiro e junho, utilizando recursos de Business Intelligence para transformar dados em informações estratégicas para tomada de decisão.

O dashboard foi construído no Power BI utilizando Power Query para tratamento dos dados e DAX para criação de métricas e indicadores de desempenho.

---

## 🎯 Objetivos da Análise

- Monitorar o faturamento das vendas ao longo do período analisado.
- Identificar os países com maior volume de receita.
- Avaliar o desempenho dos vendedores.
- Analisar a participação das companhias aéreas nas vendas.
- Entender o comportamento dos departamentos corporativos.
- Disponibilizar uma visão executiva e interativa dos dados.

## 🔄 Tratamento dos Dados

Durante o desenvolvimento do projeto foram realizadas as seguintes etapas:

- Importação de múltiplos arquivos CSV.
- Consolidação automática dos dados utilizando a funcionalidade "Combinar Arquivos".
- Padronização dos tipos de dados.
- Criação das colunas:
  - Mês
  - Número do Mês
- Organização cronológica dos períodos analisados.
- Modelagem dos dados para construção dos indicadores.
